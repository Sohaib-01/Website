Random Meal Generator

Used HTML, CSS and JS.

Project Github

Meal Generator made with HTML, CSS, JS and a public API. the public API is from a website called TheMealDB; An open, crowd-sourced database of Recipes from around the world.

I found this project really interesting as I wanted to build something really simple using an API. This project helped me to understand how a public API can be incorporated into a webpage.

I will be utilising this idea in a future app that I tend to build based on cooking different recipes with minimal ingredients.
