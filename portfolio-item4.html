Landing Page with Blur Effect

Created using HTML and CSS

This is a simple landing page with a blur effect added to it. The page is also fully responsive.

I really wanted to create a landing page with a unique effect on it, I came across this blur effect and added it to the page using HTML and CSS. When the mouse hovers on the logo and the brand name it blurs out and the background becomes clearer, when the mouse is taken away from the page then the background blurs and the logo/brand name clears up.

The blur effect is something I really like and will be using it in future projects/websites.
